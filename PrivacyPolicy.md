**Last Updated:** May 11, 2026

By using Setupify, you agree to the practices described in this Privacy Policy.

# 1. About Setupify

Setupify is a Discord utility bot designed to help users create advanced server setups, including:

- Roles
- Channels
- Categories
- Permission structures
- Server presets
- Automated configuration systems

Setupify is not affiliated with or endorsed by Discord Inc.

To provide functionality, Setupify may collect and temporarily store limited information including:

## Server Data

- Server IDs
- Channel IDs
- Role IDs
- Category IDs
- Server configuration settings
- Setup presets and templates

## User Data

- User IDs
- Command usage data
- Interaction data related to bot features

## Technical Data

- Error logs
- Diagnostic information
- Usage statistics

Setupify does NOT intentionally collect:

- Passwords
- Payment information
- Private Discord messages
- Email addresses
- Sensitive personal information
